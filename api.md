### PHP (cURL)

```php

  $ch = curl_init($url);

    "X-API-Key: $apiKey",
    "Content-Type: application/json"

  curl_close($ch);
  
  if ($httpCode !== 200 && $httpCode !== 201) {
    throw new Exception("HTTP error! status: $httpCode");
  }
  
  return json_decode($response, true);
}
```
